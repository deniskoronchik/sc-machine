## Clone repository

```sh
git clone git@github.com:ostis-dev/sc-machine.git
cd sc-machine
git submodule update --init --recursive
```

## Build on Ubuntu

### Dependencies

Install python dependencies:

__Ubuntu 18.04+__

```sh
sudo apt install python3-pip
pip3 install -r requirements.txt
```

__Ubuntu 18.04+__

```sh
sudo apt install librocksdb-dev libglib2.0-dev qtbase5-dev libboost-system-dev libboost-filesystem-dev libboost-program-options-dev make cmake antlr4 gcc g++ llvm-7 libcurl4-openssl-dev libclang-7-dev
```

### Make

```sh
mkdir build
cd build
cmake ..
make
```

## Build on OSX

### Dependencies

- Install dependencies with brew:
```
brew install boost glib llvm antlr antlr4-cpp-runtime pkgconfig rocksdb protobuf
```

- Install python 3.8
- Install python packages: 
```
pip3.8 install -r requirements.txt
```

### Make

```sh
mkdir build
cd build
cmake ..
make
```
